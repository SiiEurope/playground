Playground - Read Me

General

The project is intended to be used as a base for Java developers to demonstrate their skills in developing web applications.

Building the Project

The easiest way for building the project is to use Maven. Maven is not part of the source code.

To build the project with Maven just run mvn package in the projects root directory.

Running the Project

For running the project you need a Java web server like Apache Tomcat. The web server is not part of the source code.

The following steps describe the deployment and execution of the project for Apache Tomcat:

  1. Rename playground-1.0.war from the target project directory to playground.war
  2. Copy playground.war to the webapps directory of your Tomcat installation
  3. Run bin/startup.sh in your Tomcat installation to start the server
  4. Open a web browser and go to http://localhost:8080/playground

Source Code

The project does not use any Java web frameworks and consists only of some simple JSP's and servlets. For UI the project uses the Bootstrap library.

Important files:

src/main/java/com/sii/playground/entities/Car.java
Model class for a car.
src/main/java/com/sii/playground/entities/CarManager.java
Service class for cars. Allows to retrieve, save and delete cars.
src/main/java/com/sii/playground/web/CarServlet.java
Servlet handling GET and POST requests related to cars
src/main/webapp/index.jsp
Starting page showing a list of cars.
src/main/webapp/car.jsp
Detail page for a car.
